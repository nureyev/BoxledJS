    getDefinitionByName

    • [fullPath]
    )
    Function static

    Defined in ..\src\Utils.js:65

    This will return the class according to a String describing the classpath+name e.g.: 'boxledjs.Map' will return the actual class to be used for instatiation. This is used to reference custom classes in Tiled.

    Parameters:

    • className String

      The fully qualified name of the class. e.g.: 'boxledjs.Map'

    • [pkg=window] Object optional

      The package to start searching, defaults to 'window'

    • [fullPath] String optional

      The full path, only used for recursive iterations, not needed for the initial call.

    Function: The constructor of the class.
