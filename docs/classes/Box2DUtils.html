    Whenever you want to create a custom object that is supposed to interact with the Box2D b2World, it is strongly recommended that you call this method in the constructor of your class, see the example below:

    function Hero() {
  var self = this;

  // in the constructor or in a sub-method of the constructor
  boxledjs.Box2DUtils.injectBox2d(this, boxledjs.Box2DUtils.RECT, 100, 150, 1, 0.5, 0.05);

  // implement this method if you want to apply some special settings after
  // the body was created
  this.bxd.onBodyCreated = function(body) { self.onBodyCreated(body) };
}

Hero.prototype.onBodyCreated = function(body) {
  // for example setting a fixed rotation or more steps
  body.SetFixedRotation(true);
  body.SetBullet(true);
}

    After the injection, the object will have a property called bxd, it holds the body and a sensors object, where you can check if 'top','left','bottom' or 'right' is currently touching anything.

    Parameters:

    • object Object

      The object that the b2Body is injected to

    • type String

      The type of the body, currently only 'rect' and 'circle' are supported.

    • widthOrRadius Number

      The width of the body, or the radius in case of a circle.

    • height Number

      The height of the body (ignored if circle).

    • [density=1] Number optional

      The density of the body, this correlated to the Box2D-physics model.

    • [friction=0.5] Number optional

      The friction of the body, this correlated to the Box2D-physics model.

    • [restitution=0.05] Number optional

      The restitution of the body, this correlated to the Box2D-physics model.

    • [dynamic=true] Boolean optional

      Defines if the body is supposed to be a dynamic or static(non-moving) body.
